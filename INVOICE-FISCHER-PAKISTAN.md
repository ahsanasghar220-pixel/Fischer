## 1. PROJECT OVERVIEW

Complete design, development, and deployment of a full-stack e-commerce web application for Fischer Pakistan, a leading home appliances manufacturer. The project includes a customer-facing storefront, admin dashboard, dealer portal, and service request management system.

**Tech Stack:**
- Frontend: React 18, TypeScript, Vite, TailwindCSS, Framer Motion
- Backend: Laravel 10 (PHP 8.2), MySQL, Sanctum Auth, Spatie Permissions
- Infrastructure: Client-provided hosting & domain
